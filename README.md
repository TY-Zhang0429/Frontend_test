# Intelligent Dialogue Topic Selection Assistant

A beautiful and intelligent website for selecting dialogue topics, built with React, Vite, and Tailwind CSS.

## Features

- **Topic Analysis**: Input keywords and get detailed analysis including search index, competition, and ROI scores.
- **Zone Classification**: Results are categorized into Blue Ocean, Stable Long Tail, and Red Ocean zones.
- **Visualizations**: Clear and attractive presentation of data.
- **Real-time API**: Fetches data from a live backend service.

## Tech Stack

- React 18
- TypeScript
- Vite
- Tailwind CSS

## Getting Started

1.  **Install dependencies:**
    ```bash
    npm install
    ```

2.  **Start the development server:**
    ```bash
    npm run dev
    ```

3.  **Build for production:**
    ```bash
    npm run build
    ```
